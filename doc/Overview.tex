\section{Libraries}

These libraries model main spectral types and vary key parameters
like metallicity, effective temperature, surface gravity. They
consider photospheres with balanced Local Thermodynamical Equilibrium
conditions. Other issues like binarity, proxcimity to hot stars,
extinction etc are not in the models.


-Pickles \cite{1998PASP..110..863P} 1998 131 observed flux calibrated stellar spectra
-Kurucz 1993 models and  Kurucz and Castelli ATLAS-9 and ATLAS-12 models
-- 1993 model
-- ATLAS-9
-- ATLAS-12
- MILES library
- Others (RAVE/SDSS/APOGEE I) etc.
